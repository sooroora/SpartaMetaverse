# SpartaMetaverse

## 📖 개요
미니게임을 즐길 수 있는 나만의 공간입니다.
간단한 게임들을 가볍게 즐겨보아요!

---

## 🧩 주요 기능
- 다양한 미니게임
- 타일 맵 이동
- UnityEvent 를 활용한 상호작용

---

## 📅 개발 기간
- 25.10.22 ~ 25.10.29

---

## ⚙️ 개발 환경
- **게임엔진:** Unity 2022.3.62f2
- **언어:** C#  
- **IDE:** Rider
